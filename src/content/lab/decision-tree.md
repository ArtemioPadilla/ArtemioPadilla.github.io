Build classification and regression trees interactively using CART / ID3 algorithms. Choose split criteria (Gini impurity, information gain, variance reduction), visualize the tree structure, explore decision boundaries on 2D scatter plots, and trace predictions step by step. All implemented from scratch with zero dependencies.
